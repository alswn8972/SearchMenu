# part2: 멀티모델/가중치/성능비교 메뉴 검색 시스템

## 메뉴 데이터 준비
1. `part1/ia-data.json` 파일을 `part2/data/menu_data.json`으로 복사
2. 데이터 구조:
   ```json
   {
       "Category": "카테고리명",
       "Service": "서비스명",
       "page_name": "페이지명",
       "hierarchy": ["계층 구조"]
   }
   ```
